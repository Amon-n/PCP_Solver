# PCP_Solver
This script is designed to solve the PCP-problem as efficient as possible by giving a single or an infinte amount of answers based on a given Input. 
## What is the PCP-Problem
![pcp_problem](pictures/pcp_problem_def.png)

The Post Correspondence Problem (PCP) is a semi-decidable decision problem that asks whether, given a finite list of pairs of strings, you can select a sequence of these pairs such that the concatenation of the first components equals the concatenation of the second components.
